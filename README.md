# common-utils
公共工具类

# 使用方法
```xml
<project>
    <repositories>
        <repository>
            <id>org.gin</id>
            <name>common-utils</name>
            <url>https://agmonk.github.io/common-utils/maven-repo/</url>
        </repository>
    </repositories>
    
    <dependency>
        <groupId>org.gin</groupId>
        <artifactId>common-utils</artifactId>
        <version>1.0.0</version>
    </dependency>
</project>
```
